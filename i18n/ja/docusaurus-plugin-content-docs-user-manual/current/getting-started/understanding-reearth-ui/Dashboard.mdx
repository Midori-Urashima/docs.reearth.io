アプリケーションへのログインに成功すると、ダッシュボードに移動します。これは、アプリケーションのホームページで、いくつかの便利な機能を使用できます。

<Image path={require('./img/3-1.png').default} />

 **ワークスペースセクション**には、現在参加しているワークスペースと、メンバーのアバターが表示されます。右下の⚙️ボタンを押すと、現在のワークスペースの設定画面が表示されます（ワークスペースの詳細）。

-**プロジェクトセクション**には、現在のワークスペースにあるすべてのプロジェクト（保存されたプロ ジェクトを除く）が表示されます。

-**クイックスタート**は、簡単な操作のボタンを集めたものです。今後、さらにボタンを追加していく予定です。

- **プロジェクトカード**
<Image path={require('./img/3-2.png').default} />

プロジェクトカードにカーソルを合わせると、プロジェクトの詳細（名前、説明、公開状況）が表示されます。3つのボタンで、編集ページ、プレビューページ、プロジェクト設定ページに移動できます。

## 今後において

今後のバージョンアップでは、開発変更履歴、新機能の紹介、コミュニティからのニュースなど、他の機能モジュールを追加していく予定です。
